# 🛒 E-Commerce Sales Analysis (SQL Project)

## 📌 Project Overview

This project focuses on analyzing **E-commerce sales data using SQL** to extract meaningful business insights. The analysis answers important business questions related to **sales performance, customer behavior, revenue distribution, and seller performance**.

The goal of this project is to demonstrate strong **SQL analytical skills** by solving real-world business problems using **basic, intermediate, and advanced SQL queries**.

---

## 🎯 Objectives

* Analyze customer purchasing patterns
* Understand sales distribution across product categories
* Identify top customers and sellers
* Measure business growth over time
* Calculate retention and revenue metrics

## 📊 Business Problems Solved

### 🔹 Basic SQL Queries

These queries focus on fundamental SQL concepts such as filtering, aggregation, and grouping.

1. List all unique cities where customers are located.
2. Count the number of orders placed in 2017.
3. Find the total sales per category.
4. Calculate the percentage of orders that were paid in installments.
5. Count the number of customers from each state.

---

### 🔹 Intermediate SQL Queries

These queries involve joins, aggregations, and more detailed analysis.

1. Calculate the number of orders per month in 2018.
2. Find the average number of products per order grouped by customer city.
3. Calculate the percentage of total revenue contributed by each product category.
4. Identify the correlation between product price and the number of times a product has been purchased.
5. Calculate the total revenue generated by each seller and rank them by revenue.

---

### 🔹 Advanced SQL Queries

These queries demonstrate advanced analytical SQL skills such as window functions and time-based analysis.

1. Calculate the moving average of order values for each customer over their order history.
2. Calculate the cumulative sales per month for each year.
3. Calculate the year-over-year growth rate of total sales.
4. Calculate the customer retention rate (customers who repurchase within 6 months).
5. Identify the top 3 customers who spent the most money in each year.
